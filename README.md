# 🍽️ FrootSoot-E

A modern, responsive, and animated food e-commerce website built using **HTML, CSS, and JavaScript**. The website provides an engaging shopping experience with smooth animations, attractive UI, and multiple pages for browsing delicious food items.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 🍔 Food Menu with Categories
- 🛍️ Product Listing
- 🛒 Shopping Cart
- 💰 Indian Currency (₹) Pricing
- 🎁 Discount & Special Offers Section
- 👤 User Login Page
- 👤 User Profile Page
- 📞 Contact Page
- 💳 Checkout Page
- ✨ Smooth CSS Animations
- 📱 Mobile Friendly Design
- 🎨 Modern UI/UX
- ⚡ Fast and Lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
